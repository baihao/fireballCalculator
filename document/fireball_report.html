爆炸火球计算报告

本报告汇总了火球直径随时间、表面温度随时间、大气传输率与火球热辐射的计算方法与仿真图表。各计算分别来自本项目的 Python 模型：fireball_radius_calculator.py、fireball_temperature_calculator.py、transmissivity_calculator.py、fireball_heat_radiation_calculator.py。

2. 火球表面温度随时间

温度模型采用“左侧三次多项式 + 右侧拖曳衰减”的分段形式并在交界处进行平滑连接，输出温度单位为 K。

计算公式（时间 t 以 ms 为横轴）：

  • 上升段 0 ≤ t ≤ t0：T_up(t) = p3·t^3 + p2·t^2 + p1·t + p0（单位：°C）
  • 衰减段 t ≥ t0：T_drag(t) = A + (T0 − A)·exp(−k·(t − t0))（单位：°C）
  • 平滑连接（blend）：T(t) = (1 − S)·T_up(t) + S·T_drag(t)，其中 S(s)=3s^2−2s^3，s = clamp((t − t0)/w, 0, 1)
  • 若采用严格 C1 连接（c1 模式），在 t0 处满足 T 与 dT/dt 连续，据此确定 k。
  • 最终输出（K）：T_K(t) = T(t) + 273.15

3. 大气传输率 τ 与距离

采用经验式：τ = 1.006 − 0.1171·log10(X_H2O) − 0.02368·[log10(X_H2O)]^2 − 0.03188·log10(X_CO2) + 0.001164·[log10(X_CO2)]^2， 其中 X_CO2 = (273/Ta)·S，X_H2O = (288.651/Ta)·(760/101325)·[(RH/100)·Pw^sat(Ta)]·S。默认 Ta=297.15 K、RH=48%、Pw^sat=2987.87 Pa。

4. 火球热辐射

热通量：q(x,t)=ε·σ·T(t)^4·F(x,t)·τ(x)，其中 ε=0.9，σ=5.67×10^-8 W/(m^2·K^4)，F(x,t)=1/4(D/x)^2，τ(x) 为上节的大气传输率。累积热辐射量在 0–140 ms 时间窗内积分：H(x)=∫q(x,t)dt。

4.1 热通量随时间变化

下图展示了不同距离处热通量随时间的变化曲线。可以看出，热通量在爆炸初期（约24ms）达到峰值，然后逐渐衰减。距离越近，热通量峰值越高。

热通量统计（40%Al/Rubber材料）：

  • x = 5.0 m：最大热通量 76,350 W/m²（t = 24.0 ms），总能量 5,830 J/m²
  • x = 5.5 m：最大热通量 62,259 W/m²（t = 24.0 ms），总能量 4,754 J/m²
  • x = 6.0 m：最大热通量 51,666 W/m²（t = 24.0 ms），总能量 3,945 J/m²
